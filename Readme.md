The requirements for this code to run are:

<ul>
<li>pdf2image</li>
<li>PaddleOcr</li>
<li>Poppler</li>
<li>Numpy</li>
<li>crewai</li>
<li>streamlit</li>
</ul>

To run:
```
streamlit run file_summarizer.py
```
This app takes multiple pdf files as input and summarizes each page of the pdfs given into 1 - 2 paragraphs.
